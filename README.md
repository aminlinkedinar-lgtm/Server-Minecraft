# 🎮 Minecraft Server Manager

A web application built with Django for managing Minecraft servers.

Users can register, log in, create their own servers, edit them, and manage their personal profile.

---

## 🚀 Features

- User Registration
- User Login & Logout
- Personal Profile
- Edit Profile
- Upload Profile Picture
- Delete Account
- Create Minecraft Server
- Edit Server
- Delete Server
- Search Servers
- Sort Servers
- Pagination
- Bootstrap UI
- Django Messages
- Authentication & Authorization

---

## 🛠 Built With

- Python 3.14
- Django 6
- SQLite3
- Bootstrap 5
- HTML5
- CSS3

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/aminlinkedinar-lgtm/Server-Minecraft.git
```

Go to project

```bash
cd Server-Minecraft
```

Create virtual environment

```bash
python -m venv venv
```

Activate it

Windows

```bash
venv\Scripts\activate
```

Install packages

```bash
pip install -r requirements.txt
```

Run migrations

```bash
python manage.py migrate
```

Start server

```bash
python manage.py runserver
```
